# Guest v2 — UAT чеклист (Telegram UI)

Дата: 2026-02-28
Версия: pre-UAT

## 1) Неавторизованный гость (витрина)
- [ ] `/start` открывает витрину, а не сразу запрос контакта
- [ ] Видны кнопки: О базе / Домики и фото / Проверить даты и забронировать / FAQ / Где мы находимся / Связь / Авторизоваться
- [ ] `О базе` открывается корректно
- [ ] `Домики и фото` открывается корректно (MVP текст)
- [ ] `FAQ` открывается корректно
- [ ] `Где мы находимся` открывает текст + ссылку на карты

## 2) Авторизация
- [ ] Нажатие `Авторизоваться` просит поделиться телефоном
- [ ] Свой контакт → успешная авторизация при совпадении с бронью
- [ ] Чужой контакт → отказ
- [ ] При отсутствии брони → понятное сообщение

## 3) Авторизованный кабинет
- [ ] Показываются приоритетные кнопки: Инструкция / Связаться / Как добраться
- [ ] `Моя бронь` показывает карточку с датами/суммами
- [ ] `Оплата` показывает сумму остатка
- [ ] `Wi‑Fi` открывается
- [ ] `Правила` открываются

## 4) Оплата и чек
- [ ] `Оплата -> Отправить чек оплаты`
- [ ] Фото чека отправляется админу
- [ ] Админ может approve/reject из кнопок
- [ ] При approve: гостю приходит подтверждение, бронь становится PAID
- [ ] При reject: гостю приходит уведомление об отклонении

## 5) Инструкция по времени
- [ ] Если до заезда >24ч — инструкция закрыта с понятным текстом
- [ ] Если <=24ч — инструкция доступна

## 6) FAQ / обратная связь
- [ ] `Задать свой вопрос` предлагает выбрать категорию
- [ ] После выбора категории сообщение уходит админам
- [ ] В сообщении админам видна категория

## 7) Партнёры
- [ ] Раздел `Партнёры` доступен (если feature flag включён)
- [ ] Показан v1-каталог (текст из настроек или fallback)

## 8) Feature flags (опционально, техпроверка)
- [ ] Отключение `GUEST_FEATURE_FAQ=false` скрывает FAQ из витрины
- [ ] Отключение `GUEST_FEATURE_PARTNERS=false` скрывает Партнёров из auth-меню
- [ ] Отключение `GUEST_FEATURE_SHOWCASE_HOUSES=false` скрывает Домики и фото
