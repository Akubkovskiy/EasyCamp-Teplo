# Site Architecture Decision (final)

Дата: 2026-02-28

## Решение
- Сайт (публичный) — отдельный репозиторий и отдельный docker-контур.
- Админка — часть EasyCamp, НЕ часть сайта.
- Сайт использует API EasyCamp:
  - проверка свободных дат,
  - отправка заявки/переход в Telegram-бот.

## Что исключено
- "Сайтовая" отдельная админка (лишний дублирующий контур) — не используем.

## Доступ в админку
- По прямой ссылке в EasyCamp-контур,
- ссылка может выдаваться из бота админам.
